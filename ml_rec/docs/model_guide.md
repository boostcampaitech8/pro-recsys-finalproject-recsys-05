# RecBole 모델 가이드

이 문서는 Steam 게임 추천 시스템에서 사용된 세 가지 추천 모델(EASE, BPR, LightGCN)에 대한 설명과 학습 구성을 정리합니다.

## 1. EASE (Embarrassingly Shallow Autoencoders)

### 모델 개요

EASE는 매우 간단하면서도 강력한 선형 추천 모델입니다. 각 아이템에 대한 가중치를 학습하여 사용자-아이템 상호작용을 재구성합니다.

**특징:**
- **모델 타입:** 전통적 선형 모델 (MODEL_TYPE.TRADITIONAL)
- **입력 타입:** Pointwise (단일 샘플 단위 처리)
- **평가 방식:** Ranking 기반 평가
- **학습 파라미터:** 1개 (매우 적음)
- **연산량:** 0.0 FLOPs

### 알고리즘 원리

EASE는 사용자-아이템 상호작용 행렬 X를 선형 자동인코더로 간주하고, 다음 최적화 문제를 풀이합니다:

```
minimize: ||X - XB||²_F + λ||B||²_2
subject to: diag(B) = 0  (자기 상관 제거)
```

B의 각 열은 해당 아이템의 추천 가중치를 나타내며, 폐쇄형(closed-form) 해로 직접 계산됩니다.

#### 데이터 전처리

1. **필터링 방식:** k-core 필터링 (k=30)
   - 최소 30개의 활동을 가진 사용자와 아이템만 포함
   - 희소한 데이터의 노이즈 제거 및 데이터 품질 향상

2. **활동 기반 필터링:**
   - 사용자의 게임 소유 시간, 플레이 시간, 리뷰 등을 종합한 활동도 지표로 필터링
   - 활발한 사용자와 인기 있는 게임에 집중

3. **데이터 형식:**
   ```
   user_id, item_id, rating
   ```
   - `user_id`: Steam 사용자 ID
   - `item_id`: 게임 앱 ID
   - `rating`: 모두 1로 설정 (암시적 피드백, 사용자 소유 = 긍정)

## 2. BPR (Bayesian Personalized Ranking)

### 모델 개요

BPR은 쌍별 순위 학습(Pairwise Ranking Learning)을 기반한 신경망 기반 추천 모델입니다. 사용자가 상호작용한 아이템을 상호작용하지 않은 아이템보다 선호한다는 가정 하에 학습합니다.

**특징:**
- **모델 타입:** 일반 신경망 (MODEL_TYPE.GENERAL)
- **입력 타입:** Pairwise (긍정-부정 쌍)
- **평가 방식:** Ranking 기반 평가
- **학습 파라미터:** 5,488,448개
- **연산량:** 128.0 FLOPs

### 알고리즘 원리

BPR은 다음 최적화 목표를 가집니다:

```
maximize: Σ ln(σ(ŷ_ui - ŷ_uj))
```

여기서:
- σ: sigmoid 함수
- ŷ_ui: 사용자 u가 아이템 i를 선호할 확률
- ŷ_uj: 사용자 u가 아이템 j를 선호할 확률
- (i는 상호작용한 아이템, j는 부정 샘플)

## 3. LightGCN (Light Graph Convolutional Network)

### 모델 개요

LightGCN은 그래프 신경망(GNN) 기반의 현대적 추천 모델입니다. 사용자-아이템 이분 그래프에서 그래프 컨볼루션을 통해 사용자와 아이템의 임베딩을 학습합니다.

**특징:**
- **모델 타입:** 일반 신경망 (MODEL_TYPE.GENERAL)
- **입력 타입:** Pairwise (긍정-부정 쌍)
- **평가 방식:** Ranking 기반 평가
- **학습 파라미터:** 17,441,280개
- **그래프 레이어:** 2층
- **연산량:** 0.0 FLOPs

### 알고리즘 원리

LightGCN은 사용자-아이템 이분 그래프를 다음과 같이 구성합니다:

```
그래프 구조:
User1 ─┬─ Item1
       ├─ Item2
User2 ─┼─ Item3
       └─ Item4

그래프 컨볼루션:
e_u^(l+1) = σ(Σ_{i∈N_u} 1/√|N_u||N_i| e_i^(l))
e_i^(l+1) = σ(Σ_{u∈N_i} 1/√|N_u||N_i| e_u^(l))
```

최종 임베딩은 모든 레이어의 출력을 평균내어 생성됩니다.

## 결론

- **EASE**는 간단하면서도 매우 효과적인 모델으로, 특히 k-core 필터링된 데이터에서 우수한 성능을 보입니다.
- **BPR**은 임베딩 기반의 균형잡힌 모델로, 중간 규모 데이터셋에 적합합니다.
- **LightGCN**은 그래프 신경망 기반의 최신 모델로, 충분한 계산 리소스가 있을 경우 최고의 성능을 기대할 수 있습니다.

실제 추천 시스템 구성 시에는 **계산 시간, 메모리, 성능 요구사항**을 종합적으로 고려하여 모델을 선택해야 합니다.
