📖 문서 위치 및 용도

1. README.md (진입점)
   └─ 프로젝트 개요, 빠른 시작, 기본 설정
   └─ 처음 읽어야 할 파일

2. MODELS_GUIDE.md (모델 이해)
   └─ EASE, BPR, LightGCN 상세 설명
   └─ 모델 선택 가이드
   └─ 성능 비교

3. scripts/PIPELINE.md (실행 가이드)
   └─ 데이터 전처리 → 모델 학습 → 추론
   └─ 전체 워크플로우 설명
   └─ 트러블슈팅

4. API_EXAMPLES.md (배포 가이드)
   └─ Flask/FastAPI/Docker 구현
   └─ 웹 서비스 예시
   └─ 성능 최적화 및 모니터링

5. eda/steam_inter_eda.ipynb (데이터 분석)
   └─ 데이터 탐색 및 분석
   └─ 전처리 이해

1단계: README.md 읽기 (5분)
  → 전체 프로젝트 이해

2단계: MODELS_GUIDE.md 읽기 (10분)
  → 모델 선택 및 이해

3단계: scripts/PIPELINE.md 읽기 (15분)
  → 실행 방법 학습

4단계: 코드 실행
  → 실제 동작 확인

5단계: API_EXAMPLES.md 읽기 (배포 예정시)
  → 웹 서비스 구현
